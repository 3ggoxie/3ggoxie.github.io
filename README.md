# 川渝7日游个人攻略（人均消费约3K）

## 简介
本攻略为个人制作的从宁波出发川渝地区7日游行程，适合想要深度体验川渝文化、美食与自然风光的旅行者。人均消费约为3000元，具体费用因个人选择而异。以下为详细行程安排。

### **行程总览（3.4-3.11）**
**路线调整**：成都（3天）→ 都江堰（1天）→ 重庆（3天）  
**交通衔接**：成都东站→重庆北站（高铁1.5小时），重庆江北机场返宁波  
**核心亮点**：成都人文+熊猫基地→重庆魔幻夜景+山城文化
